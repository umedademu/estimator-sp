# アプリ概要

## 目的

本プロジェクトの目的は、土木・建設業務の見積作成をスマートフォンで完結できるWebアプリを開発すること。  
現場や移動中でも、入力・計算・確認を短時間で実行できる操作体験を実現する。

## 想定ユーザー

- 現場担当者
- 見積作成担当者
- 工事計画・積算に関わる管理者

## 提供価値

- スマホ画面に最適化された見積入力
- 入力値に連動する計算結果の即時表示
- 計算根拠（式）の可視化による確認性向上
- 外出先での迅速な見積判断

## 機能スコープ

- 2次製品入力
- 擁壁入力
- 舗装入力
- 撤去工事入力

各入力ページで、主要な数量・単価・金額を計算結果として表示し、項目ごとの計算ロジックを確認できる構成とする。

## 設計方針

- モバイルファーストのUI
- 片手操作を前提にした入力導線
- 入力ミスを抑える項目設計（選択式・数値入力ガイド）
- 計算ロジックの一貫性と説明可能性を重視

## ゴール

- スマートフォンのみで見積作業の主要フローが完結すること
- 各入力ページで、計算結果と計算根拠を同じ画面で確認できること
- 実務で再利用しやすい、拡張可能な構成にすること

# 撤去工事入力ページ「計算結果」カラム 計算式メモ

## 前提

- 「切り上げ」は `Math.ceil` を使用
- m単価は「金額 ÷ 距離」で算出
- 0除算になる場合は 0 扱い

## 基本計算

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 搬出廃材量 | 距離 × 幅 × 厚さ × 2.4 で算出します。 |
| 施工面積 | 距離 × 幅で算出します。 |

## カッター工事

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| カッター延長 | カッターピッチがある場合は `（距離÷ピッチ）×（幅÷ピッチ）`、未入力時は外周 `（距離×2）+（幅×2）`、0入力時は0mとします。 |
| カッターm単価 | 選択したカッター材料の単価を使用します。 |
| カッター総額 | カッターm単価 × カッター延長で算出します。 |

## 撤去工事

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 4時間積込量 | バックホー容量 × 480 で算出します。 |
| 撤去必要日数 | 搬出廃材量 ÷ 4時間積込量 を切り上げます。 |
| 撤去必要人数 | 撤去必要日数 × 4人で算出します。 |
| 1日必要機械台数 | 施工面積 ÷ 4時間積込量 ÷ 撤去必要日数で算出し、表示時に切り上げます。 |
| 機械金額1 | 選択したバックホーの単価を使用します。 |
| 機械金額2 | 選択した使用機種の単価を使用します。 |
| 機械総金額 | （機械金額1 + 機械金額2）× 撤去必要日数で算出します。 |
| 掘削施工金額 | （撤去必要人数 × 25,000円）+ 機械総金額で算出します。 |
| 撤去工事m施工単価 | 掘削施工金額 ÷ 距離で算出します。 |

## 廃材搬出

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 搬出廃材日数 | 搬出廃材量 ÷ 4時間積込量 を切り上げます。 |
| 車両積載量 | 選択した残土搬出車両の積載量を使用します。 |
| 搬出台数 | 搬出廃材量 ÷ 車両積載量 を切り上げます。 |
| 搬出車両単価 | 選択した残土搬出車両の単価を使用します。 |
| 常用ダンプ台数 | 搬出台数 ÷ 1日ダンプの回転数 ÷ 搬出廃材日数 で算出します。 |
| 常用ダンプ単価 | 選択した常用ダンプの単価を使用します。 |
| 生コン単価 | 選択した生コンの単価を使用します。 |
| 廃材搬出金額 | （搬出廃材日数 × 車両積載量 × 搬出台数 × 生コン単価）+（常用ダンプ単価 × 常用ダンプ台数）で算出します。 |
| 撤去m単価見積 | 廃材搬出金額 ÷ 距離で算出します。 |

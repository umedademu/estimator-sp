# 舗装入力ページ「計算結果」カラム 計算式メモ

## 前提

- 「切り上げ」は端数を上の整数に上げる処理
- m単価・m2単価は「合計金額 ÷ 対象数量」で算出
- 0除算になる場合は 0 扱い

## 土工事

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 掘削幅 | 幅 + 0.4m で算出 |
| 施工面積 | 距離 × 幅で算出 |
| 掘削高さ | 舗装or土間厚さ + 上層砕石厚さ + 下層砕石厚さで算出 |
| 切盛高低差 | 現況の高さ - 計画の高さ + 掘削高さで算出 |
| 掘削土量 | 切盛高低差 × 掘削幅 × 距離 × 1.25 で算出 |
| 4時間掘削量 | 設定係数 × 機械容量で算出 |
| 掘削日数 | 掘削土量 ÷ 4時間掘削量 を切り上げ |
| 掘削人数 | 掘削日数 × 4人 で算出 |
| 掘削人件費 | 掘削人数 × 人区で算出 |
| 1日機械台数 | 掘削土量 ÷ 4時間掘削量 ÷ 掘削日数 を切り上げ |
| 機械単価 | 選択した使用機械の単価を使用 |
| 機械金額 | 機械単価 × 1日機械台数 × 掘削日数で算出 |
| 掘削施工金額 | 掘削人件費 + 機械金額で算出 |
| 土工事単価／m | 掘削施工金額 ÷ 距離で算出 |

## 上層砕石

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 上層砕石量 | 距離 × 掘削幅 × 上層砕石厚さ × 1.2（ロス補正）で算出 |
| 上層砕石資材費 | 上層砕石量 × 上層砕石単価で算出 |
| 4時間敷均量 | 設定係数 × 機械容量で算出 |
| 上層砕石人数 | 施工面積に応じた設定テーブルから参照 |
| 上層砕石人件費 | 上層砕石人数 × 人区で算出 |
| 上層砕石日数 | 上層砕石量 ÷ 4時間敷均量 を切り上げ |
| 砕石機械台数 | 施工面積が100m2超かどうかの条件で決定し |
| 上層砕石機械費 | 機械単価 × 砕石機械台数 × 上層砕石日数で算出 |
| 上層砕石総額 | 上層砕石資材費 + 上層砕石人件費 + 上層砕石機械費で算出 |
| 上層砕石工事単価／m | 上層砕石総額 ÷ 距離で算出 |

## 下層砕石

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 下層砕石量 | 距離 × 掘削幅 × 下層砕石厚さ × 1.2（ロス補正）で算出 |
| 下層砕石資材費 | 下層砕石量 × 下層砕石単価で算出 |
| 4時間敷均量 | 設定係数 × 機械容量で算出 |
| 下層砕石人数 | 下層砕石量に応じた設定テーブルから参照 |
| 下層砕石人件費 | 下層砕石人数 × 人区で算出 |
| 下層砕石日数 | 下層砕石量 ÷ 4時間敷均量 を切り上げ |
| 砕石機械台数 | 施工面積が100m2超かどうかの条件で決定し |
| 下層砕石機械費 | 機械単価 × 砕石機械台数 × 下層砕石日数で算出 |
| 下層砕石総額 | 下層砕石資材費 + 下層砕石人件費 + 下層砕石機械費で算出 |
| 下層砕石工事単価／m | 下層砕石総額 ÷ 距離で算出 |

## 残土搬出

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 搬出残土量 | 掘削土量をそのまま搬出残土量として扱う |
| 搬出日数 | 搬出残土量 ÷ 4時間掘削量 を切り上げ |
| ダンプ容積 | 選択した残土搬出車両の容積を使用 |
| 搬出台数 | 搬出残土量 ÷ ダンプ容積 を切り上げ |
| 搬出車両単価 | 選択した残土搬出車両の単価を使用 |
| 常用ダンプ台数 | 搬出台数 ÷ 5 を切り上げ |
| 常用ダンプ単価 | 選択車両に対応する常用ダンプ単価を使用 |
| 残土搬出人件費 | 搬出日数 × 人区で算出 |
| 残土搬出金額 | （搬出台数 × 搬出車両単価）+（常用ダンプ台数 × 常用ダンプ単価）+ 残土搬出人件費 で算出 |
| 残土搬出単価／m | 残土搬出金額 ÷ 距離で算出 |

## 不陸調整

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 不陸調整 | 施工面積に応じた設定テーブルから人数を参照 |
| m²不陸整正単価 | （不陸調整人数 × 人区）÷ 施工面積で算出 |
| 不陸調整材料量 | 施工面積 × 0.03t/m2 で算出 |
| 不陸調整機械台数 | 不陸調整材料量が 0 より大きければ 1 台、0 なら 0 台 |

## 舗装層

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 舗装1層目材料単価 | 選択した舗装1層目材料の単価を使用 |
| 舗装2層目材料単価 | 選択した舗装2層目材料の単価を使用 |
| 舗装1層目重量 | 施工面積 × 舗装1層目厚さ × 2.4 × 1.1 で算出 |
| 舗装2層目重量 | 施工面積 × 舗装2層目厚さ × 2.4 × 1.1 で算出 |
| 舗装1層目人数 | 舗装1層目重量に応じた設定テーブルから参照 |
| 舗装2層目人数 | 舗装2層目重量に応じた設定テーブルから参照 |
| 1層目乳剤量 (m²/L) | 施工面積 × 1L/m2 で算出 |
| 1層目乳剤資材費 | 1層目乳剤量 × 450円/L + 5,000円（諸経費）で算出 |
| 1層目乳剤単価 | 1層目乳剤資材費 ÷ 1層目乳剤量で算出 |
| 1層目乳剤人数 | 施工面積に応じた設定テーブルから参照 |
| 1層目乳剤人件費 | 1層目乳剤人数 × 人区で算出 |
| 2層目乳剤量 | 施工面積 × 0.6L/m2 で算出 |
| 2層目乳剤金額 | 2層目乳剤量 × 450円/L + 5,000円（諸経費）で算出 |
| 2層目乳剤単価 | 2層目乳剤金額 ÷ 2層目乳剤量で算出 |

## 鉄筋

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 鉄筋主筋本数 | （距離 - 0.12）÷ 主筋ピッチで算出 |
| 鉄筋配力筋本数 | （幅 - 0.12）÷ 配力筋ピッチで算出 |
| 鉄筋重量 | 重さ係数 ×（主筋本数×有効距離 + 配力筋本数×有効幅）で算出 |
| 鉄筋労務費 | 鉄筋重量が1t未満なら50,000円固定、1t以上は `鉄筋重量 × 50,000円 × 係数` で算出 |
| スペーサー費 | 鉄筋重量 × 3,000円/t で算出 |
| 加工費 | 鉄筋重量 × 15,000円で算出 |
| 運賃 | 鉄筋重量 × 9,000円で算出 |
| 鉄筋諸費用 | スペーサー費 + 加工費 + 運賃で算出 |
| 鉄筋材料費 | 鉄筋重量 × 鉄筋単価 × 1.3 で算出 |
| 鉄筋単価／t | 鉄筋材料費 ÷ 鉄筋重量で算出 |

## メッシュ

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| メッシュ枚数 | 施工面積 ÷ 2 ÷ 0.656m2 で算出 |
| メッシュ金額 | メッシュ枚数 × メッシュ単価で算出 |
| メッシュ人数 | メッシュ枚数 ÷ 60 で算出 |

## コンクリート・型枠

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 生コン体積 | 土間厚み × 施工面積で算出 |
| 生コン単価 | 選択した生コンの単価を使用 |
| 打設人数 | 施工面積が40m2未満は2人、80m2未満は3人、それ以上は `施工面積 ÷ 40` で算出 |
| ポンプ台数 | 打設人数が 7人未満=1台、12人未満=2台、20人未満=3台、それ以上=4台 |
| 型枠距離 | （距離 × 2）+（幅 × 2）で算出 |
| 型枠本数 | 型枠距離 × 2 ÷ 4m で算出 |
| 型枠金額 | 型枠距離 × 型枠単価で算出 |
| 型枠人数 | 型枠距離 ÷ 30 で算出 |
| 型枠人件費 | 型枠人数 × 人区で算出 |

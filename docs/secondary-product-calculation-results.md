# 2次製品入力ページ「計算結果」カラム 計算式メモ

## 前提

- 「切り上げ」は端数を上の整数に上げる処理、「切り捨て」は端数を下の整数に落とす処理
- `/m` の単価は「金額 ÷ 距離」
- 距離が 0 の場合、`/m` 系の値は 0 扱い

## 掘削

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 掘削幅 | 製品幅に 0.4m を加算して掘削幅を算出 (`掘削幅 = 製品幅 + 0.4`) |
| 掘削高さ | 現況高から計画高を引き、製品高・砕石厚・ベース厚を加算して掘削高さを算出 (`掘削高さ = (現況高 - 計画高) + 製品高 + 砕石厚 + ベース厚`) |
| 掘削土量 | 掘削幅・距離・掘削高さに土量変化率 1.25 を掛けて掘削土量を算出 (`掘削土量 = 掘削幅 × 距離 × 掘削高さ × 1.25`) |
| 4時間掘削量 | 設定の4時間掘削量係数に、使用機械名から抽出した容量値を乗算。機械未選択時は0 (`4時間掘削量 = 係数 × 機械容量`) |
| 掘削日数 | 掘削土量を4時間掘削量で割り、端数を切り上げて掘削日数を算出 |
| 掘削1日人数 | 「距離÷15」に2人を足し、端数を切り上げて1日あたり人数を算出 |
| 掘削人数 | 掘削日数と掘削1日人数を掛けて、延べ人数を算出 (`掘削人数 = 掘削日数 × 掘削1日人数`) |
| 機械単価 | 選択した使用機械の単価をそのまま使用 |
| 機械金額 | 機械単価に「埋め戻し日数 + 掘削日数」を乗算 (`機械金額 = 機械単価 × (埋め戻し日数 + 掘削日数)`) |
| 掘削施工金額 | 掘削人数×労務単価に機械金額を加算 (`掘削施工金額 = (掘削人数 × 労務単価) + 機械金額`) |
| 掘削単価/m | 掘削施工金額を距離で割って m あたり単価を算出 (`掘削単価/m = 掘削施工金額 ÷ 距離`) |

## 残土搬出

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 搬出残土量 | 掘削土量から埋め戻し量を引いて、搬出する残土量を算出 (`搬出残土量 = 掘削土量 - 埋め戻し量`) |
| 搬出日数 | 搬出残土量を4時間掘削量で割り、端数を切り上げて搬出日数を算出 |
| ダンプ容積 | 選択した残土搬出車両の容量をそのまま使用 |
| 搬出台数 | 搬出残土量をダンプ容積で割り、端数を切り上げて必要台数を算出 |
| 搬出車両単価 | 選択車両名を機械単価テーブルから参照して単価を決定（完全一致→部分一致→トン数推定→既定値）。 |
| 常用ダンプ台数 | 搬出台数を「4時間掘削量÷2」で割り、端数を切り上げて常用ダンプ台数を算出 |
| 常用ダンプ単価 | 使用機械単価の 2 倍を常用ダンプ単価とする (`常用ダンプ単価 = 機械単価 × 2`) |
| 残土搬出金額 | 搬出車両費・常用ダンプ費・搬出労務費の合計 (`残土搬出金額 = (搬出日数 × 搬出台数 × 搬出車両単価) + (常用ダンプ台数 × 常用ダンプ単価) + (搬出日数 × 労務単価)`) |
| 残土搬出単価/m | 残土搬出金額を距離で割って m あたり単価を算出 (`残土搬出単価/m = 残土搬出金額 ÷ 距離`) |

## 埋め戻し

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 埋め戻し量 | 距離・製品幅・製品高に 1.25 を掛けて埋め戻し量を算出 (`埋め戻し量 = 距離 × 製品幅 × 製品高 × 1.25`) |
| 埋め戻し日数 | 埋め戻し量を4時間掘削量で割り、端数を切り上げて日数を算出 |
| 埋め戻し人数 | 「1 + 埋め戻し量÷2」の端数を切り捨てて人数を算出 |
| 人件費 | 埋め戻し人数に労務単価を掛けて人件費を算出 (`人件費 = 埋め戻し人数 × 労務単価`) |

## 砕石関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 砕石量 | 距離・砕石厚・掘削幅に 1.2 を掛けて砕石量を算出 (`砕石量 = 距離 × 砕石厚 × 掘削幅 × 1.2`) |
| 砕石人数 | 「距離×掘削幅」を9で割り、端数を切り上げて人数を算出 |
| 砕石日数 | 「砕石量 ÷ 4時間掘削量 ÷ 砕石人数」に1.4を掛け、端数を切り上げて日数を算出 |
| 砕石人件費 | 砕石日数・砕石人数・労務単価を乗算 (`砕石人件費 = 砕石日数 × 砕石人数 × 労務単価`) |
| 砕石機械費 | 砕石日数に機械単価を乗算 (`砕石機械費 = 砕石日数 × 機械単価`) |
| 砕石施工金額 | 砕石人件費と砕石機械費の合計 (`砕石施工金額 = 砕石人件費 + 砕石機械費`) |
| 砕石資材費 | 砕石量に砕石単価を乗算 (`砕石資材費 = 砕石量 × 砕石単価`) |
| 砕石総額 | 砕石施工金額と砕石資材費の合計 (`砕石総額 = 砕石施工金額 + 砕石資材費`) |
| 砕石工事単価/m | 砕石総額を距離で割って m あたり単価を算出 (`砕石工事単価/m = 砕石総額 ÷ 距離`) |

## ベース関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| ベース幅 | 製品幅に 0.1m を加算してベース幅を算出 (`ベース幅 = 製品幅 + 0.1`) |
| ベース生コン量 | 距離・ベース幅・ベース厚に 1.1 を掛けて生コン量を算出 (`ベース生コン量 = 距離 × ベース幅 × ベース厚 × 1.1`) |
| 生コン単価(m3) | 選択した生コンの単価をそのまま使用 |
| 打設人数 | 生コン量に応じた人数テーブルで決定（0:0人、0.5未満:2人、1.5未満:3人、2未満:4人、3未満:5人、4未満:6人、7未満:7人、9未満:9人、それ以上:10人）。 |
| 型枠面積 | 距離にベース厚と両側分の2を乗算 (`型枠面積 = 距離 × ベース厚 × 2`) |
| 型枠資材費 | 型枠面積に型枠単価を乗算 (`型枠資材費 = 型枠面積 × 型枠単価`) |
| ベース合計金額 | 生コン材料費・打設労務費・型枠資材費の合計 (`ベース合計金額 = (ベース生コン量 × 生コン単価) + (打設人数 × 労務単価) + 型枠資材費`) |
| ベース単価/m | ベース合計金額を距離で割って m あたり単価を算出 (`ベース単価/m = ベース合計金額 ÷ 距離`) |

## 二次製品関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| モルタル | ベース幅・距離・0.02・段数を掛けてモルタル量を算出 (`モルタル量 = ベース幅 × 距離 × 0.02 × 段数`) |
| 砂 | モルタル量に製品長を掛けて砂量を算出（製品長が0なら1として扱う）。 (`砂量 = モルタル量 × 製品長`) |
| 砂金額 | 砂量に砂単価を乗算 (`砂金額 = 砂量 × 砂単価`) |
| セメント | 「モルタル量 × 0.3 × 1.6 × 1000 ÷ 25」の結果を切り上げ、袋数を算出 |
| セメント金額 | セメント袋数に600円/袋を乗算 (`セメント金額 = セメント袋数 × 600`) |
| 水 | モルタル量に `0.1 × 1000` を掛けて水量(L)を算出 (`水量 = モルタル量 × 0.1 × 1000`) |
| 2次製品単価 | 選択した二次製品の単価をそのまま使用 |
| 本数 | `距離÷製品長×段数` の小数部が 0.49 以下なら「切り捨て+2本」、それ以外は切り上げで本数を決定（製品長が0なら1として扱う）。 |
| 材料費合計 | 本数×2次製品単価に運搬費を加算 (`材料費合計 = (本数 × 2次製品単価) + 運搬費`) |
| 据付人数 | 本数に「0.03 × 施工係数」を掛け、端数を切り上げて据付人数を算出 |
| 2次製品設置合計 | 砂金額・セメント金額・材料費合計・据付労務費の合計 (`2次製品設置合計 = 砂金額 + セメント金額 + 材料費合計 + (据付人数 × 据付単価)`) |
| 2次製品設置単価/m | 2次製品設置合計を距離で割って m あたり単価を算出 (`2次製品設置単価/m = 2次製品設置合計 ÷ 距離`) |

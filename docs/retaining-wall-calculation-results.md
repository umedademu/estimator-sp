# 擁壁入力ページ「計算結果」カラム 計算式メモ

## 前提

- 「切り上げ」は `Math.ceil` を使用
- m単価・m2単価は「合計金額 ÷ 対象数量」で算出
- 0除算になる場合は 0 扱い

## 掘削

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 掘削幅 | 底版幅に 1.0m を加えて算出します。 |
| 設計構造物高 | 竪壁高さ・底版厚・捨てコン厚・砕石厚さを合算します。 |
| 掘削高さ | （現況高 - 設計高）に設計構造物高を加算します。 |
| 掘削土量 | 掘削幅 × 距離 × 掘削高さ × 1.25（土量変化率）で算出します。 |
| 掘削日数 | 掘削土量 ÷ 4時間掘削量 を切り上げます。 |
| 4時間掘削量 | 設定係数 × 機械性能で算出します。 |
| 掘削人数 | 掘削日数 × 3人/日 で算出します。 |
| 掘削人件費 | 掘削人数 × 労務単価で算出します。 |
| 機械単価 | 選択した使用機械の単価を採用します。 |
| 掘削機械費 | 機械単価 × 掘削日数で算出します。 |
| 掘削合計 | 掘削人件費 + 掘削機械費で算出します。 |

## 残土搬出

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 搬出残土量 | 掘削土量 - 埋め戻し量で算出します。 |
| ダンプ容積 | 選択した残土搬出車両の容量を使用します。 |
| 搬出台数 | 搬出残土量 ÷ ダンプ容積 を切り上げます。 |
| 搬出車両単価 | 選択した残土搬出車両の単価を使用します。 |
| 搬出車両金額 | 搬出車両単価 × 搬出台数で算出します。 |
| 常用ダンプ | 搬出台数に応じた常用ダンプ費用テーブルから算出します。 |
| 残土搬出人数 | 搬出残土量 ÷ 4時間掘削量 を切り上げます。 |
| 残土搬出人件費 | 残土搬出人数 × 労務単価で算出します。 |
| 残土搬出合計 | 搬出車両金額 + 残土搬出人件費 + 常用ダンプ費用で算出します。 |

## 埋め戻し・土工事合計

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 埋め戻し量 | 掘削土量 - 構造物体積 - 砕石量で算出します。 |
| 埋め戻し日数 | 埋め戻し量 ÷ 4時間掘削量 を切り上げます。 |
| 埋め戻し人数 | 埋め戻し日数 × 3人/日 で算出します。 |
| 埋め戻し人件費 | 埋め戻し人数 × 労務単価で算出します。 |
| 埋戻機械費 | 機械単価 × 埋め戻し日数で算出します。 |
| 機械金額 | 掘削機械費 + 埋戻機械費で算出します。 |
| 埋め戻し合計 | 埋め戻し人件費 + 埋戻機械費で算出します。 |
| 埋め戻しm2単価 | 埋め戻し合計 ÷（距離 × 底版幅）で算出します。 |
| 土工事合計 | 掘削合計 + 残土搬出合計 + 埋め戻し合計で算出します。 |
| 土工事m2単価 | 土工事合計 ÷（距離 × 底版幅）で算出します。 |

## 砕石関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 砕石量 | 掘削幅 × 距離 × 砕石厚さ × 1.2（砕石変化率）で算出します。 |
| 砕石人数 | 掘削面積 ÷ 12m2/人 を切り上げます。 |
| 砕石日数 | （砕石量 ÷ 4時間掘削量 × 1.4）を切り上げます。 |
| 砕石人件費 | 砕石人数 × 砕石日数 × 労務単価で算出します。 |
| 砕石機械費 | 機械単価 × 砕石日数で算出します。 |
| 砕石施工金額 | 砕石人件費 + 砕石機械費で算出します。 |
| 砕石資材費 | 砕石単価 × 砕石量で算出します。 |
| 砕石総額 | 砕石施工金額 + 砕石資材費で算出します。 |
| 砕石工事単価/m | 砕石総額 ÷（距離 × 底版幅）で算出します。 |

## 型枠

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 壁型枠面積 | 壁両面の面積式で算出します（距離・縦壁高さ・壁厚を使用）。 |
| 底版型枠面積 | 底版両面の面積式で算出します（距離・底版幅・底版厚を使用）。 |
| 底版型枠代 | 底版型枠面積 × 底版型枠m2単価で算出します。 |
| 壁型枠代 | 壁型枠面積 × 壁型枠m2単価で算出します。 |
| 型枠代合計 | 壁型枠代 + 底版型枠代で算出します。 |
| m単価 | 型枠代合計 ÷ 距離で算出します。 |
| 型枠距離 | （距離 × 2）+（底版幅 × 2）で算出します。 |
| 型枠本数 | 型枠距離 × 2 ÷ 4m を切り上げます。 |
| 型枠金額 | 型枠距離 × 型枠単価/m で算出します。 |
| 型枠人数 | 型枠距離 ÷ 30m を切り上げます。 |
| 型枠人件費 | 型枠人数 × 労務単価で算出します。 |

## コンクリート関連

### 縦壁CO関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 縦壁体積 | 距離 × 縦壁高さ × 縦壁厚で算出します。 |
| 縦壁CO金額 | 縦壁体積 × CO単価で算出します。 |
| 縦壁人数 | （距離 × 縦壁高さ ÷ 縦壁人工/m2）を切り上げます。 |
| 縦壁人件費 | 縦壁人数 × コンクリート人区で算出します。 |
| 縦壁COm単価 | 縦壁CO金額 ÷ 距離で算出します。 |

### 底版CO関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 底版体積 | 距離 × 底版幅 × 底版厚で算出します。 |
| 底版CO金額 | 底版体積 × CO単価で算出します。 |
| 底版人数 | （距離 × 底版幅 ÷ 底版人工/m2）を切り上げます。 |
| 底版人件費 | 底版人数 × コンクリート人区で算出します。 |
| 底盤COm単価 | 底版CO金額 ÷ 距離で算出します。 |

### 捨てコン関連

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 捨てコン体積 | 距離 ×（底版幅 + 捨コン出幅）× 捨てコン厚で算出します。 |
| 捨てコン人数 | （距離 × 底版幅 ÷ 捨てコン人工/m2）を切り上げます。 |
| 捨てコン人件費 | 捨てコン人数 × コンクリート人区で算出します。 |
| 捨てコンCO金額 | 捨てコン体積 × CO単価で算出します。 |
| 捨てCOm単価 | 捨てコンCO金額 ÷ 距離で算出します。 |

### コンクリート総合

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| CO単価 | 選択した生コンの単価を使用します。 |
| CO体積合計 | 壁体積 + 底版体積 + 捨てコン体積で算出します。 |
| CO合計 | 壁CO金額 + 底版CO金額 + 捨てコンCO金額で算出します。 |
| 人件費合計 | 壁人件費 + 底版人件費 + 捨てコン人件費で算出します。 |
| 全体合計 | CO合計 + 人件費合計で算出します。 |
| 全体m単価 | 全体合計 ÷ 距離で算出します。 |

## 鉄筋

注記: 一部ラベルは選択鉄筋規格に応じて `D10S重量` のように動的表示されます。

| 項目 | ホバー表示向けの自然言語式 |
|---|---|
| 縦主筋 | （距離 - 縦壁かぶり厚×2）÷ 縦壁ピッチ を切り上げます。 |
| 縦主筋長さ | 縦壁高さ - 縦壁かぶり厚×2 で算出します。 |
| 縦配力筋 | 縦主筋長さ ÷ 縦壁ピッチ を切り上げます。 |
| 底版主筋 | （距離 - 底版かぶり厚×2）÷ 底版ピッチ を切り上げます。 |
| 底版主筋長さ | 距離 - 底版かぶり厚×2 で算出します。 |
| 底版配力筋 | 底版配力筋長さ ÷ 底版ピッチ を切り上げます。 |
| 底版配力筋長さ | 底版幅 - 底版かぶり厚×2 で算出します。 |
| 底版有効長さ | 距離 - 底版かぶり厚×2 で算出します。 |
| {鉄筋規格}重さ係数 | 選択した鉄筋規格の重さ係数を使用します。 |
| {鉄筋規格}重量 | 主筋・配力筋の各本数と長さの合計に 1.3 を掛け、重さ係数を乗じて算出します。 |
| {鉄筋規格}人区 | 鉄筋重量の切り上げ値 × 人区単価で算出します。 |
| {鉄筋規格}小計 | 鉄筋人区 +（鉄筋重量 × 鉄筋単価 × 1.3）で算出します。 |
| {鉄筋規格}単価/t | 鉄筋小計 ÷ 鉄筋重量で算出します。 |
| {鉄筋規格}単価/m | 鉄筋小計 ÷ 距離で算出します。 |
